# Data, DS, and ML Training Plan

This repository is now used as a technical training track.

## How We Work
For each step:
1. Goal
2. One command block
3. Why this matters
4. Expected output
5. Validation check

Do not move to next step until output is reviewed.

## Module A - Data Analysis with SQL (current)
Session 1: Data understanding and business metrics
Session 2: Aggregations and segment diagnostics
Session 3: CTE and window functions for trend/risk
Session 4: Decision memo from SQL evidence

## Module B - Data Management and Quality
Session 5: Data quality tests (nulls, duplicates, anomalies)
Session 6: Basic data modeling (fact and dimensions)
Session 7: Reproducible pipeline habits

## Module C - Data Science Foundations
Session 8: Problem framing (target, features, leakage)
Session 9: Baseline model and train/test split
Session 10: Error analysis and model comparison

## Module D - Machine Learning Practice
Session 11: Feature engineering and validation
Session 12: Model metrics and trade-offs
Session 13: Business impact interpretation

## Module E - Interview Readiness
Session 14: 30-second and 2-minute explanation drills
Session 15: Technical Q&A simulation with project artifacts
